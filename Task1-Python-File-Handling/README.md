# Python File Handling & Automation

## Project Overview
This project is a Python-based file organizer that automatically sorts files into folders based on their file extensions.

## Features
- Detects file types automatically
- Creates folders dynamically
- Moves files into categorized folders
- Supports multiple file formats

## Technologies Used
- Python 3
- OS Module
- Shutil Module
- VS Code
- GitHub

## Supported File Types

| Category | Extensions |
|----------|-------------|
| Images | .jpg, .jpeg, .png |
| PDFs | .pdf |
| Videos | .mp4 |
| Audio | .mp3 |
| Text | .txt |
